# ToDo App (Flutter + Hive)

A simple and beautiful **ToDo List App** built with **Flutter** and **Hive (NoSQL Database)** for local storage.  
This app allows users to **add, delete, and mark tasks as completed**.  

---

##  Features 😁
- Add new tasks ✍️  
- Mark tasks as completed ✅  
- Delete tasks 🗑️  
- Data persistence using **Hive Database** 📂  
- Simple and clean **UI design** 🎨  

---

##  Tech Stack 🛠️
- **Flutter** (Frontend UI)  
- **Dart** (Programming Language)  
- **Hive & hive_flutter** (Local Database)  
- **flutter_slidable** (Swipe to delete feature)
